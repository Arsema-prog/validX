# Changelog

All notable changes to this project will be documented in this file.

The format is based on **Keep a Changelog**
and this project follows **Semantic Versioning**.

---

## [1.0.0] - 2025-01-XX

### Added

- Number validators: `isNumber`, `isPositive`, `isInRange`
- String validators: `isString`, `minLength`, `maxLength`
- Security validators: `isEmail`, `isPhone`, `isStrongPassword`
- Initial test suite using Vitest
- TypeScript support with type definitions
